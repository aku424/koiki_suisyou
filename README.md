## nodeバージョン
v20.3.0

## npm script
| スクリプト | 説明 |
| ---- | ---- |
| npm run start| jsファイルの監視開始|
| npm run build| jsファイルコンパイル|
| npm run build:img| src配下のimgディレクトリを公開ディレクトリに圧縮＋複製、webp生成 |


## ディレクトリ

| ディレクトリ | 説明 |
| ---- | ---- |
| src| 開発用ディレクトリ、ファイル格納|
| src/dev| 開発用Scss,Javascript格納|
| src/images| 開発用の圧縮、webp生成、複製前の画像素材格納|
